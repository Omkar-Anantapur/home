About the Course

The Design and Analysis of Algorithms (DAA) course provides a strong foundation for solving computational problems efficiently. It develops analytical skills, optimization techniques, and problem-solving strategies vital to software design and system efficiency.

Project Overview

“City Design” explores algorithmic applications in smart urban systems. It simulates data structures for route optimization, traffic flow, and urban planning using computational logic. This project bridges theoretical computer science with real-world design applications.
