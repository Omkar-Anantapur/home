Episode 1 – REAL-TIME AQI PREDICTION & POLLUTION MONITORING

What It Means

This system continuously measures air quality across the city and predicts future AQI values using historical sensor data.

Why It Is Necessary

Air pollution causes respiratory diseases and reduces life expectancy. Early AQI prediction allows governments to restrict traffic, regulate factories, and issue public health alerts in advance.

C++ Code (Dynamic Input)

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<double> aqi(n);

    for(int i = 0; i < n; i++)
        cin >> aqi[i];

    double sx=0, sy=0, sxy=0, sxx=0;
    for(int i = 0; i < n; i++) {
        sx += i;
        sy += aqi[i];
        sxy += i * aqi[i];
        sxx += i * i;
    }

    double b = (n*sxy - sx*sy) / (n*sxx - sx*sx);
    double a = (sy - b*sx) / n;

    int h;
    cin >> h;

    cout << "Predicted AQI: " << a + b*(n+h) << endl;
}

Algorithm & Code Working

Uses Linear Regression to model AQI trend and predict future value in O(n) time. The code computes regression coefficients and forecasts AQI for the requested future hour.

Sensors / Input Source

  • PM2.5 & PM10 Laser Sensors (PMS5003, SDS011)
  • Gas Sensors (MQ-135, MQ-7)
  • Data sent via ESP32 + WiFi/GSM to server → C++ program reads from DB/CSV.

Episode 2 – SMART INDUSTRIAL EMISSION VIOLATION DETECTION

What It Means

This system monitors factory chimney gases and detects abnormal pollution release automatically.

Why It Is Necessary

Illegal industrial emissions are a major contributor to urban air pollution. Automated detection prevents night-time violations and protects public health.

C++ Code

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<double> e(n);
    for(int i = 0; i < n; i++) cin >> e[i];

    double mean = accumulate(e.begin(), e.end(), 0.0) / n;
    double var = 0;
    for(double x : e) var += (x-mean)*(x-mean);

    double sd = sqrt(var/n);

    for(int i = 0; i < n; i++)
        if(fabs(e[i] - mean) > 3 * sd)
            cout << "Violation at sensor " << i << endl;
}

Algorithm & Code Working

Uses Z-Score Anomaly Detection to flag emission spikes beyond 3σ. Runs in O(n) time.

Sensors / Input

  • MQ-7 (CO), MQ-136 (SO₂), NO₂ electro-chemical sensors
  • Data via GSM to pollution control server

Episode 3 – URBAN HEAT ISLAND & GREEN COVER OPTIMIZATION

What It Means

Identifies extreme heat zones in the city and suggests green cover locations.

Why It Is Necessary

High urban temperatures increase electricity load and cause heat strokes. Green planning reduces city temperature naturally.

C++ Code

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;

    vector<double> temp(n);
    for(int i = 0; i < n; i++) cin >> temp[i];

    vector<double> cent(k);
    for(int i = 0; i < k; i++) cent[i] = temp[i];

    vector<int> lbl(n);

    for(int it = 0; it < 20; it++) {
        for(int i = 0; i < n; i++) {
            double best = 1e9;
            for(int j = 0; j < k; j++)
                if(abs(temp[i]-cent[j]) < best) {
                    best = abs(temp[i]-cent[j]);
                    lbl[i] = j;
                }
        }

        vector<double> sum(k), cnt(k);
        for(int i = 0; i < n; i++) {
            sum[lbl[i]] += temp[i];
            cnt[lbl[i]]++;
        }

        for(int j = 0; j < k; j++)
            cent[j] = sum[j]/cnt[j];
    }

    int hot = max_element(cent.begin(), cent.end()) - cent.begin();
    cout << "Hottest Cluster: " << hot << endl;
}

Algorithm & Code Working

Uses K-Means Clustering to group city zones by temperature. Complexity O(n·k·iter).

Sensors / Input

  • Thermal sensors
  • Satellite data (Landsat/MODIS)
  • Weather stations

Episode 6 – SMART NOISE POLLUTION CONTROL

What It Means

Monitors sound levels in sensitive zones like hospitals and schools.

Why It Is Necessary

Noise pollution causes stress, sleep disorders, and heart problems.

C++ Code

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<double> s(n);
    for(int i = 0; i < n; i++) cin >> s[i];

    double sum = 0;
    for(double x : s) sum += x*x;

    double rms = sqrt(sum/n);

    if(rms > 0.6)
        cout << "Noise Violation\n";
    else
        cout << "Normal\n";
}

Algorithm & Code Working

Uses RMS (Root Mean Square) Energy Detection in O(n) time.

Sensors / Input

  • SPL / MEMS microphone sensors
  • Data fed every second

Episode 9 – POLLUTION-AWARE ROUTE FOR SCHOOLS & HOSPITALS

What It Means

Finds least-polluted paths for ambulances and school buses.

Why It Is Necessary

Reduces harmful exposure to children and elderly.

C++ Code

#include <bits/stdc++.h>
using namespace std;

int main() {
    int V, E;
    cin >> V >> E;

    vector<vector<pair<int,int>>> g(V);
    for(int i = 0; i < E; i++) {
        int u,v,w;
        cin >> u >> v >> w;
        g[u].push_back({v,w});
        g[v].push_back({u,w});
    }

    int s, d;
    cin >> s >> d;

    vector<int> dist(V, 1e9);
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;

    dist[s] = 0;
    pq.push({0,s});

    while(!pq.empty()) {
        auto [cd,u] = pq.top(); pq.pop();
        for(auto &e : g[u]) {
            if(dist[e.first] > cd + e.second) {
                dist[e.first] = cd + e.second;
                pq.push({dist[e.first], e.first});
            }
        }
    }

    cout << "Min Pollution Path: " << dist[d] << endl;
}

Algorithm & Code Working

Uses Dijkstra’s Algorithm with O(E log V) complexity.

Input Source

  • Road network
  • Live AQI per road
  • GPS traffic data
