BUSINESS CASE 1 — Smart Billboard Advertisement Allocation Optimizer

1. What it means

Optimizes the allocation of digital billboards to advertisers to maximize revenue while ensuring fair exposure time.

2. Why it is necessary

  • Prevents overbooking of popular billboards
  • Ensures advertisers get fair time share
  • Maximizes city revenue
  • Helps schedule thousands of ads daily

3. Decision Flow

Ad Requests → Sort by Profit/Duration → Greedy Slot Selection → Final Schedule

4. C++ Code

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, T;
    cin >> n >> T;
    vector<pair<double,int>> ads; // {profit per time, time}

    for(int i=0;i<n;i++){
        double p; int t;
        cin >> p >> t;
        ads.push_back({p/t, t});
    }

    sort(ads.rbegin(), ads.rend());

    int used = 0;
    for(auto &a : ads){
        if(used + a.second <= T){
            used += a.second;
        }
    }
    cout << "Used time: " << used;
}

5. Algorithm Used

Greedy interval scheduling based on profit/time ratio.

6. How code works

Sort ads by profitability per second, keep selecting until time runs out.

7. Input Sources

Advertising platform logs, billboard availability database.

BUSINESS CASE 2 — Municipal Document Digitization Load Balancer

1. What it means

Distributes scanning workload among servers so no server is overloaded.

2. Why it is necessary

  • Large cities scan millions of documents
  • Prevents server slowdown
  • Ensures timely digitization

3. Decision Flow

Document Sizes → Min-Heap of Servers → Assign to Lightest Loaded

4. C++ Code

#include <bits/stdc++.h>
using namespace std;

int main(){
    int n,m; cin>>n>>m;
    vector<int> docs(n);
    for(int &x: docs) cin>>x;

    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
    for(int i=0;i<m;i++) pq.push({0,i});

    for(int x : docs){
        auto [load,id] = pq.top(); pq.pop();
        load += x;
        pq.push({load,id});
    }

    auto res = pq.top();
    cout << "Max load on server: " << res.first;
}

5. Algorithm Used

Greedy load balancing with min-heap.

6. How code works

Always assigns next document to the server with lowest current load.

7. Input Sources

Document sizes, server performance data.

BUSINESS CASE 3 — Public Feedback Clustering & Issue Prioritizer

1. What it means

Automatically groups similar citizen complaints and ranks the most common issues.

2. Why it is necessary

  • Prevents backlog
  • Helps the city know common problems
  • Automates manual sorting

3. Decision Flow

TF-IDF → K-Means Clustering → Rank by Cluster Size → Priority List

4. C++ Code (Simplified cluster sizing)

#include <bits/stdc++.h>
using namespace std;

int main(){
    int n,k; cin>>n>>k;
    vector<int> cluster(n);
    for(int &x: cluster) cin >> x;

    vector<int> count(k,0);
    for(int x: cluster) count[x]++;

    int best = max_element(count.begin(), count.end()) - count.begin();
    cout << "Top priority cluster: " << best;
}

5. Algorithm Used

K-means clustering + frequency ranking.

6. How code works

Counts how many complaints belong to each cluster, picks biggest cluster.

7. Input Sources

Citizen app feedback, social media text mining.

BUSINESS CASE 4 — Festival Crowd Movement Simulation & Exit Optimization

1. What it means

Simulates crowd flow during festivals to identify safe exit points.

2. Why it is necessary

  • Prevents stampedes
  • Optimizes emergency exits
  • Helps police plan crowd control

3. Decision Flow

Grid Map → BFS Flow Simulation → Detect Bottlenecks → Add/Shift Exits

4. C++ Code

#include <bits/stdc++.h>
using namespace std;

int main(){
    int n; cin>>n;
    vector<string> grid(n);
    for(auto &s:grid) cin>>s;

    queue<pair<int,int>> q;
    vector<vector<int>> dist(n, vector<int>(n, -1));

    q.push({0,0});
    dist[0][0]=0;

    int dx[4]={1,-1,0,0}, dy[4]={0,0,1,-1};

    while(!q.empty()){
        auto [x,y]=q.front(); q.pop();
        for(int k=0;k<4;k++){
            int nx=x+dx[k], ny=y+dy[k];
            if(nx>=0 && ny>=0 && nx<n && ny<n && grid[nx][ny]!='#' && dist[nx][ny]==-1){
                dist[nx][ny]=dist[x][y]+1;
                q.push({nx,ny});
            }
        }
    }

    cout << "Max crowd distance: " << dist[n-1][n-1];
}

5. Algorithm Used

BFS for crowd-flow simulation.

6. How code works

Calculates how long it takes for crowd to reach exit; larger values mean bottlenecks.

7. Input Sources

CCTV heatmaps, pedestrian sensors.

BUSINESS CASE 5 — Drone Battery Swapping Station Optimization

1. What it means

Finds best places to install drone battery swap stations for delivery drones.

2. Why it is necessary

  • Reduces drone downtime
  • Ensures large coverage area
  • Critical for e-governance drone operations

3. Decision Flow

Drone Routes → K-Median Clustering → Final Station Coordinates

4. C++ Code (Simplified center selection)

#include <bits/stdc++.h>
using namespace std;

double dist(pair<int,int>a,pair<int,int>b){
    return hypot(a.first-b.first, a.second-b.second);
}

int main(){
    int n; cin>>n;
    vector<pair<int,int>> pts(n);
    for(auto &p:pts) cin>>p.first>>p.second;

    pair<int,int> center = pts[rand()%n];

    double total=0;
    for(auto &p:pts) total+=dist(center,p);

    cout<<"Cost: "<<total;
}

5. Algorithm Used

k-median approximation (center selection heuristic).

6. How code works

Chooses a center and computes distance sum (station cost).

7. Input Sources

Drone route data from city delivery logs.

BUSINESS CASE 6 — Smart Emergency Broadcast Channel Optimization

1. What it means

Selects minimal set of communication channels (SMS, WhatsApp, FM, sirens) to reach maximum population during emergencies.

2. Why it is necessary

  • Ensures timely alerts
  • Reduces panic
  • Prevents network overload

3. Decision Flow

Population Coverage Data → Greedy Selection → Enough Coverage?

4. C++ Code

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    vector<int> reach(n);
    for(int &x : reach) cin >> x;

    int target; cin >> target;
    priority_queue<pair<int,int>> pq;

    for(int i=0;i<n;i++) pq.push({reach[i], i});

    int covered=0;
    while(!pq.empty() && covered < target){
        auto [r,id] = pq.top(); pq.pop();
        covered += r;
        cout << "Channel " << id << "\n";
    }
}

5. Algorithm Used

Greedy set cover approximation.

6. How code works

Repeatedly picks channel covering max people.

7. Input Sources

Telecom coverage maps, app activity data.

BUSINESS CASE 7 — City Library Dynamic Book Rebalancing System

1. What it means

Optimizes distribution of popular books across libraries so demand is balanced.

2. Why it is necessary

  • Prevents shortages of popular books
  • Reduces logistical cost
  • Improves citizen satisfaction

3. Decision Flow

Demand per Library → Min-Cost Flow → Move Books

4. C++ Code (Simplified supply-demand balancing)

#include <bits/stdc++.h>
using namespace std;

int main(){
    int n; cin>>n;
    vector<int> need(n), have(n);

    for(int &x: need) cin>>x;
    for(int &x: have) cin>>x;

    sort(need.begin(), need.end());
    sort(have.begin(), have.end());

    int moves=0;
    for(int i=0;i<n;i++)
        moves += abs(need[i] - have[i]);

    cout << "Books to move: " << moves;
}

5. Algorithm Used

Greedy matching (sorted arrays).

6. How code works

Matches inventory to demand minimizing movement.

7. Input Sources

Library book popularity data.

BUSINESS CASE 8 — Smart Disaster Inventory Positioning

1. What it means

Places emergency kits across the city to minimize worst-case response time.

2. Why it is necessary

  • Saves lives during disasters
  • Reduces travel time for rescue teams
  • Ensures equitable access

3. Decision Flow

Binary Search on Radius → BFS Coverage Check → Feasible?

4. C++ Code

#include <bits/stdc++.h>
using namespace std;

bool feasible(int r, vector<int>& pts){
    for(int x : pts) if(x > r) return false;
    return true;
}

int main(){
    int n; cin>>n;
    vector<int> pts(n);
    for(int &x:pts) cin>>x;

    int l=1, r=100000, ans=-1;
    while(l<=r){
        int mid=(l+r)/2;
        if(feasible(mid, pts)) ans=mid, r=mid-1;
        else l=mid+1;
    }

    cout<<"Min radius: "<<ans;
}

5. Algorithm Used

Binary search for minimax radius.

6. How code works

Finds smallest r that covers all demand points.

7. Input Sources

Disaster risk maps, warehouse locations.

BUSINESS CASE 9 — Smart Heatwave Shelter Capacity Planner

1. What it means

Ensures heatwave shelters can safely accommodate predicted population.

2. Why it is necessary

  • Prevents deaths, overcrowding, and chaos.

3. Decision Flow

Sort Demands + Capacities → Binary Search → Feasibility Match

4. C++ Code

#include <bits/stdc++.h>
using namespace std;

bool canAssign(vector<int>& pop, vector<int>& cap, int limit){
    int i=0,j=0;
    while(i<pop.size() && j<cap.size()){
        if(pop[i] <= limit && cap[j] >= pop[i]) i++, j++;
        else j++;
    }
    return i==pop.size();
}

int main(){
    int n,m; cin>>n>>m;
    vector<int> pop(n), cap(m);

    for(int &x: pop) cin>>x;
    for(int &x: cap) cin>>x;

    sort(pop.begin(), pop.end());
    sort(cap.begin(), cap.end());

    int l=1,r=100000,ans=-1;
    while(l<=r){
        int mid=(l+r)/2;
        if(canAssign(pop,cap,mid)) ans=mid, r=mid-1;
        else l=mid+1;
    }
    cout<<"Max people per shelter: "<<ans;
}

5. Algorithm Used

Binary search + greedy matching.

6. How code works

Finds smallest max-capacity limit that allows safe assignment.

7. Input Sources

IMD heatwave forecast, shelter database.
