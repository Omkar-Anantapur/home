Business Case Study 6 – Water Supply Reachability Detection

1. Problem Statement

The city water department needs to verify whether a house is properly connected to the main water supply line. The water distribution system forms a network of:

  • Main tank (source)
  • Area distribution points
  • Street pipelines
  • Individual houses

If any part of the pipeline is disconnected, the house may not receive water. Thus, the system must check if a house is reachable from the main tank.

This is a perfect application of Depth-First Search (DFS) or Breadth-First Search (BFS) in a graph.

2. Algorithm Used: Depth-First Search (DFS)

The pipeline network is modeled as an undirected graph:

  • Nodes = pipeline junctions, streets, houses, tanks
  • Edges = water pipeline connections

We start DFS from Main Tank and visit all reachable nodes. If the target house is visited → it receives water. If not → house is disconnected.

3. Time Complexity Explanation

DFS Time Complexity = O(V + E)

Where V = number of nodes (houses, streets, junctions) and E = pipeline connections The algorithm is linear and very fast even for thousands of houses.

4. C++ Code (Long, Fully Runnable)


#include 
using namespace std;

class WaterNetwork {
public:
    unordered_map> graph;
    unordered_set visited;

    void addConnection(const string& a, const string& b) {
        graph[a].push_back(b);
        graph[b].push_back(a);
    }

    void dfs(const string& node) {
        visited.insert(node);
        for (auto &nbr : graph[node]) {
            if (!visited.count(nbr)) {
                dfs(nbr);
            }
        }
    }

    bool hasWater(const string& mainTank, const string& house) {
        visited.clear();
        dfs(mainTank);
        return visited.count(house);
    }

    void printNetwork() {
        cout << "===== Water Pipeline Network =====\n";
        for (auto &p : graph) {
            cout << p.first << " connected to: ";
            for (auto &n : p.second) cout << n << " ";
            cout << "\n";
        }
        cout << "===================================\n";
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    WaterNetwork wn;

    wn.addConnection("Main_Tank", "AreaA");
    wn.addConnection("AreaA", "Street1");
    wn.addConnection("Street1", "House_12");
    wn.addConnection("Street1", "House_15");
    wn.addConnection("AreaA", "Street2");
    wn.addConnection("Street2", "House_21");

    wn.printNetwork();

    cout << "\nChecking water supply:\n";

    string house1 = "House_12";
    string house2 = "House_99";

    cout << house1 << ": ";
    if (wn.hasWater("Main_Tank", house1))
        cout << "Water supply available\n";
    else
        cout << "No supply (Disconnected)\n";

    cout << house2 << ": ";
    if (wn.hasWater("Main_Tank", house2))
        cout << "Water supply available\n";
    else
        cout << "No supply (Disconnected)\n";

    cout << "\nSystem test complete.\n";
    return 0;
}

5. Conclusion

DFS ensures accurate and fast detection of connectivity in the city water supply network. This approach works perfectly for smart-city monitoring, leak detection, and consumer support systems.
