Business Case Study 5 – Real-Time Parking Slot Availability

1. Problem Statement

Urban parking areas face heavy traffic. Drivers waste time circling around looking for a free spot. The city authorities want a system that shows real-time available slots in each parking lot:

  • When a vehicle enters → free slots decrease.
  • When a vehicle exits → free slots increase.
  • Users must instantly check free slots through the app.

The system must handle thousands of updates every minute across the city.

2. Algorithm Used: Hash Map Direct-Access Operations

Every parking lot has an ID (string). We store free slots as key → value pairs: 

"Hubballi_Central" → 54  
"Railway_Parking" → 12  
"Airport_Parking" → 89

Using a hash map:

  • Insert lot → O(1)
  • Update slot count → O(1)
  • Query free slots → O(1)

This is the fastest possible structure for this problem.

3. Time Complexity Explanation

All operations (average) = O(1)

Even with thousands of parking lots, real-time updates are fast. Worst-case can be O(n), but practically hash maps behave in constant time.

4. C++ Code (Long, Realistic, Copyable)


#include 
using namespace std;

class ParkingSystem {
private:
    unordered_map lotSlots;
    unordered_map lotCapacity;

public:

    void addLot(const string& lot, int capacity) {
        lotSlots[lot] = capacity;
        lotCapacity[lot] = capacity;
    }

    bool parkCar(const string& lot) {
        if (lotSlots.find(lot) == lotSlots.end()) {
            cout << "Error: Unknown parking lot.\n";
            return false;
        }
        if (lotSlots[lot] > 0) {
            lotSlots[lot]--;
            return true;
        }
        return false;
    }

    void removeCar(const string& lot) {
        if (lotSlots.find(lot) != lotSlots.end()) {
            if (lotSlots[lot] < lotCapacity[lot]) {
                lotSlots[lot]++;
            }
        }
    }

    int getFreeSlots(const string& lot) {
        if (lotSlots.find(lot) == lotSlots.end()) return -1;
        return lotSlots[lot];
    }

    void printLots() {
        cout << "===== Current Parking Lots =====\n";
        for (auto &p : lotSlots) {
            cout << p.first << " : " 
                 << p.second << "/" 
                 << lotCapacity[p.first] << " free\n";
        }
        cout << "=================================\n";
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    ParkingSystem ps;

    ps.addLot("Hubballi_Central", 100);
    ps.addLot("Railway_Parking", 50);
    ps.addLot("Airport_Parking", 200);

    ps.printLots();

    ps.parkCar("Hubballi_Central");
    ps.parkCar("Hubballi_Central");
    ps.parkCar("Railway_Parking");

    cout << "\nAfter parking operations:\n";
    ps.printLots();

    ps.removeCar("Hubballi_Central");
    ps.removeCar("Airport_Parking");

    cout << "\nAfter exit operations:\n";
    ps.printLots();

    cout << "\nQuery free slots:\n";
    cout << "Hubballi_Central: " << ps.getFreeSlots("Hubballi_Central") << "\n";
    cout << "Airport_Parking: " << ps.getFreeSlots("Airport_Parking") << "\n";

    return 0;
}

5. Conclusion

A hash map-based parking availability system gives instant updates and real-time monitoring. It is scalable for large cities and perfectly fits smart-city automation models.
